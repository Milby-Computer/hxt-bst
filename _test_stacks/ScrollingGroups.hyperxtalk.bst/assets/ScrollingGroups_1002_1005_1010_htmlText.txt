<p>Moves the scrollWheel script message dispatch and the findGroupUnderPoint group-hit logic into shared infrastructure so all three platforms benefit from the same behaviour.</p>
<p>MCCard::findGroupUnderPoint(x, y) replaces the macOS-only static helper in desktop.cpp, making the group-under-mouse lookup available to all platform files via card.h. The Windows (WM_MOUSEWHEEL / WM_MOUSEHWHEEL) and Linux (GDK scroll events) handlers in w32dcw32.cpp and lnxdclnx.cpp are updated to use findGroupUnderPoint and send scrollWheel before falling through to the existing kdown behavior, exactly mirroring what macOS already does.</p>
<p>The Linux GDK direction mapping is also corrected in the process  GDK_SCROLL_UP/DOWN/LEFT/RIGHT now produce signed 1 deltas consistent with the macOS and Windows convention rather than the previously inverted mapping.</p>
