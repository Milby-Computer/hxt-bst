<p><b><font face="verdana" size="11">A note about entries designated as "Unlimited":</font></b></p>
<p>Since each open <a><font face="verdana" size="11">stack file</font></a><font face="verdana" size="11"> resides completely in memory, Revolution <a>stacks</a></font><font face="verdana" size="11"> (and all structures within a <a>stack</a></font><font face="verdana" size="11">) are effectively limited by available memory and by Revolution's total address space of 4G (4,294,967,296 <a>bytes</a></font><font face="verdana" size="11">) on 32-bit systems, or 16P (18,446,744,073,709,551,616 <a>bytes</a></font><font face="verdana" size="11">) on 64-bit systems.</font></p>
