<p><b><font face="verdana" size="11">Notes:</font></b></p>
<p>To <a><font face="verdana" size="11">import</font></a> a picture file in a supported <a><font face="verdana" size="11">format</font></a> and make it an <a><font face="verdana" size="11">image</font></a> <a><font face="verdana" size="11">object</font></a>, use the <a><b><font face="verdana" size="11">import</font></b></a> <a><font face="verdana" size="11">command</font></a>. To <a><font face="verdana" size="11">export</font></a> a picture file, use the <a><b><font face="verdana" size="11">export</font></b></a> <a><font face="verdana" size="11">command</font></a>.</p>
<p></p>
<p><b><font face="verdana" size="11" color="#404040"><img src="200926" char=" "> Note:</font></b>  Because of patent issues, exporting a <a><font face="verdana" size="11">GIF</font></a> file requires a special license. If your application needs to <a><font face="verdana" size="11">export</font></a> in this format, contact Runtime Revolution to make arrangements for a <a><font face="verdana" size="11">GIF</font></a> export key.</p>
<p></p>
<p>To find out how an <a><font face="verdana" size="11">image</font></a> is stored internally, check the <a><font face="verdana" size="11">image&rsquo;s</font></a> <a><b><font face="verdana" size="11">paintCompression</font></b></a> <a><font face="verdana" size="11">property</font></a>.</p>
<p></p>
<p>To display a <font face="verdana" size="11">picture file</font> without <a><font face="verdana" size="11">importing</font></a> it into the <a><font face="verdana" size="11">stack</font></a>, create an empty <a><font face="verdana" size="11">image</font></a> and set its <a><b><font face="verdana" size="11">filename</font></b></a> <a><font face="verdana" size="11">property</font></a> to the <a><font face="verdana" size="11">file&rsquo;s</font></a> <a><font face="verdana" size="11">name and location</font></a>.</p>
<p></p>
<p><b><font face="verdana" size="11" color="#404040"><img src="202651" char=" "> Tip:</font></b>  Some picture <a><font face="verdana" size="11">formats</font></a> (such as <font face="verdana" size="11">TIFF</font>) that are not supported by the <a><font face="verdana" size="11">image</font></a> <a><font face="verdana" size="11">object</font></a> can be displayed instead by <a><font face="verdana" size="11">QuickTime</font></a>. To display a picture using <a><font face="verdana" size="11">QuickTime</font></a>, you create a <a><font face="verdana" size="11">player</font></a> instead of an <a><font face="verdana" size="11">image</font></a>, then set the <a><font face="verdana" size="11">player&rsquo;s</font></a> <a><b><font face="verdana" size="11">filename</font></b></a>, to display the picture file.</p>
