<p><font face="Lucida Grande" size="12" color="#000000">If all goes well, you should have grabbed the RunRev logo from the top of our website and displayed it on your card. Cool huh?</font></p>
<p></p>
<p><b><font face="Lucida Grande" size="14" color="#990006">The example application</font></b></p>
<p><font face="Lucida Grande" size="12" color="#000000">For an example of this I will show you how to create a simple application that will grab weather information (satellite images, cloud cover etc.) from the BBC weather website and display it in your application.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">Although this application looks quite complex the key parts of it are very straightforward.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">All we need do is create a URL of the correct format that the BBC uses for its images and then set the filename of the main image to that URL.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">You can try the application out by typing:</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">go stack URL "http://www.runrev.com/newsletter/october/issue34/WeatherStation.rev"</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">into your Revolution message box.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">(It should be noted that some images are not available for certain parts of the world, or for certain times. For example, the Visible image is only available for every 3 hour slot for places outside the UK.)</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">The BBC weather site URL structure</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">The BBC weather site uses the following structure for the image files on its web servers.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12">http://www.bbc.co.uk/weather/charts/ </font><font face="Lucida Grande" size="12" color="#000000">/__.jpg</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">Where:</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">is the main area of the world you are interested in (e.g. UK, World)</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">narrows down the area to a more specific location ( e.g. NS (North Scotland), NE (North England)</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">is the type of chart you are interested in ( e.g. Rain, Cloud)</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">is a simple string representing the date and hour you want. It is formatted as YYMMDDHH</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">Now we know the structure of the URL used to retrieve an image from the BBC web server, we need to write some code to create a correctly formatted URL.</font></p>
<p></p>
<p><b><font face="Lucida Grande" size="14" color="#990006">The URL and image code</font></b></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">We have two selection fields - the image type and the area. Each line of these fields contains information about the image we want to retrieve.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">For the image type each line has a description followed by a bar ("|") and then the text that represents the element of the URL.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">For the area field, each line has three elements separated by a bar ("|"). Firstly there is a description of the area, second we have the text representing the URL element and lastly we have the text representing the element.</font></p>
