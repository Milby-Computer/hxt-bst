<p><b><font face="helvetica" size="12">Discussion:</font></b></p>
<p>The <font face="Courier" size="11">menuPick</font><font face="helvetica" size="12"> message is sent when the user chooses a menu item from the menu. Remember that in Revolution, menus are usually implemented as buttons:  the button's </font><font face="Courier" size="11">style</font><font face="helvetica" size="12"> and </font><font face="Courier" size="11">menuMode</font><font face="helvetica" size="12"> properties control how it is displayed, and the button's </font><font face="Courier" size="11">text</font><font face="helvetica" size="12"> property is used as the list of menu items. When the </font><font face="Courier" size="11">menuPick</font><font face="helvetica" size="12"> message is sent to the button, it triggers this handler.</font></p>
<p></p>
<p>The <font face="Courier" size="11">menuHistory</font><font face="helvetica" size="12"> property of a button menu is the number of the last-chosen menu item. Each line of the button contents becomes a separate menu item when the menu is displayed, so the </font><font face="Courier" size="11">menuHistory</font><font face="helvetica" size="12"> is also the line number of the menu item. The </font><font face="Courier" size="11">menuPick</font><font face="helvetica" size="12"> handler gets the text of the chosen menu item as a parameter. However, in this example, we use </font><i><font face="Lucida Grande" size="11">chunk expressions</font></i><font face="helvetica" size="12"> to change the contents of the button, so it's easier to deal with the line number instead.</font></p>
<p></p>
<p>When a menu item is checkmarked, the characters "!c" are placed at the start of that menu item in the button contents. For example, if the second menu item has a checkmark, the second line of the button contents starts with "!c". You checkmark a menu item by adding "!c" at the start of the corresponding line, and remove the checkmark by deleting these characters. The next time the menu is displayed, the checkmarks are shown as specified by the button contents.</p>
<p></p>
<p>This example handler tests whether the chosen menu item is already checkmarked--that is, whether its line starts with "!c". If it is, the first two characters are deleted to remove the checkmark. Otherwise, the "!c" marker is added to the start of the line to add a checkmark.</p>
<p></p>
<p><b><font face="Lucida Grande" size="11">Note:</font></b><font face="helvetica" size="12">  In a cascading menu, the process of checkmarking a menu item works the same way:  the line corresponding to the checkmarked menu item starts with "!c". The characters "!c" must precede the tab character (that indicates the menu item is part of a cascading menu).</font></p>
<p></p>
<p>This example allows multiple menu items to be checkmarked. Choosing an already-checkmarked item unchecks it again. But suppose the menu items are mutually exclusive, and we want only one menu item to be checkmarked at a time? In this case, we would remove any existing checkmark before checkmarking the new menu item. The simplest way to do this is to uncheck every menu item before checkmarking the chosen one:</p>
<p></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#980517">repeat</font><font face="Courier" size="11"> </font><font face="Courier" size="11" color="#980517">with</font><font face="Courier" size="11"> x = 1 to the </font><font face="Courier" size="11" color="#FF0000">number</font><font face="Courier" size="11"> of lines in </font><font face="Courier" size="11" color="#F88017">me</font></p>
<p><font face="Courier" size="11">    </font><font face="Courier" size="11" color="#980517">if</font><font face="Courier" size="11"> char 1 to 2 of line x of </font><font face="Courier" size="11" color="#F88017">me</font><font face="Courier" size="11"> is "!c"</font></p>
<p><font face="Courier" size="11">    </font><font face="Courier" size="11" color="#980517">then</font><font face="Courier" size="11"> </font><font face="Courier" size="11" color="#0000FF">delete</font><font face="Courier" size="11"> char 1 to 2 of line x of </font><font face="Courier" size="11" color="#F88017">me</font></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#980517">end</font><font face="Courier" size="11"> </font><font face="Courier" size="11" color="#980517">repeat</font></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#0000FF">put</font><font face="Courier" size="11"> "!c" before line myItemNumber of </font><font face="Courier" size="11" color="#F88017">me</font><font face="Courier" size="11"> </font><font face="Courier" size="11" color="#68228B">-- checkmark the item</font></p>
<p></p>
<p>The method above is simple but inefficient, since it requires the handler to loop through every menu item to find the one menu item that's checkmarked. For most menus, the difference in speed will not be noticeable, but for very complex cascading menus that may have hundreds of menu items, it may make sense to use a faster method. One alternative is to store the line number of the checkmarked menu item in a custom property of the button, and then retrieve the custom property and uncheck just that menu item:</p>
<p></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#0000FF">put</font><font face="Courier" size="11"> the currentCheckedLine of </font><font face="Courier" size="11" color="#F88017">me</font><font face="Courier" size="11"> into oldLine</font></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#0000FF">delete</font><font face="Courier" size="11"> char 1 to 2 of line oldLine of </font><font face="Courier" size="11" color="#F88017">me</font></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#0000FF">put</font><font face="Courier" size="11"> "!c" before line myItemNumber of </font><font face="Courier" size="11" color="#F88017">me</font></p>
<p><font face="Courier" size="11">  </font><font face="Courier" size="11" color="#0000FF">set</font><font face="Courier" size="11"> the currentCheckedLine of </font><font face="Courier" size="11" color="#F88017">me</font><font face="Courier" size="11"> to myItemNumber</font></p>
