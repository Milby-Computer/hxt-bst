<p><b><font face="Lucida Grande" size="14" color="#990006">STAGE 2</font></b></p>
<p><font face="Lucida Grande" size="12" color="#000000">The effect is generated using the import snapshot function with the resultant image being set as the new window shape. We are going to set blend level of the background image and then take a snapshot of the entire group. The image produced will contain alpha blended regions as well as solid areas where our controls have been placed.</font></p>
