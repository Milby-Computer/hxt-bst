<p><b><font face="Lucida Grande" size="14" color="#990006">RSS and Revolution</font></b></p>
<p><font face="Lucida Grande" size="12" color="#000000">RSS feeds can be used within Revolution using the revXML library. This provides all the functionality required to parse XML files and makes using RSS feeds incredibly easy. The library has functions to create XML trees and retrieve branches and leaves from within it.</font></p>
<p></p>
<p><font face="Lucida Grande" size="12" color="#000000">Below is a simple explanation of how to create a stack which retrieves and displays RSS feeds.  To find out more about revXML please refer to the XML section of the current User Guide.</font></p>
