<p><b><font face="helvetica" size="12">Discussion:</font></b></p>
<p>This example uses four handlers in two scripts to provide a set of draggable color swatches. Clicking a swatch and dragging it to an object--in this demo, a graphic--changes that object's color.</p>
<p></p>
<p>The example is based on <font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> property. The </font><font face="Courier" size="11">dragData</font><font face="helvetica" size="12"> contains whatever is currently being dragged--plain text, styled text, image data, or files--stored in an </font><i><font face="Lucida Grande" size="11">array</font></i><font face="helvetica" size="12"> whose keys are the data types that you can drag and drop. When the user starts the drag, you set </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> to whatever is being dragged. When the drag ends, you use the contents of </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> to perform the appropriate drop action. This example stores the color of the swatch in the "text" element of </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12">. When the user drops the swatch onto the "ball" graphic, the graphic's </font><font face="Courier" size="11">backgroundColor</font><font face="helvetica" size="12"> is set to the color name stored in </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12">.</font></p>
<p></p>
<p>The <font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> message is sent whenever the user clicks, to whatever object was clicked. In the demo, each swatch is a graphic, and the graphics have been placed in a group. The </font><font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> handler is placed in the group's script so it doesn't have to be repeated in each graphic's script.</font></p>
<p></p>
<p>When the <font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> message is sent to a swatch, it is passed on to the group, where it triggers the </font><font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> handler. This </font><font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> handler sets </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> property, which specifies the data to be dragged and dropped. For instance, if you are dragging text, that text is the contents of </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12">. In this example, the handler sets the "text" element of </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> to the name of the swatch the user clicked. This is the data that will be </font><i><font face="Lucida Grande" size="11">received</font></i><font face="helvetica" size="12"> when the swatch is dropped onto an object.</font></p>
<p></p>
<p>This <font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> handler also sets the swatch's </font><font face="Courier" size="11">borderWidth</font><font face="helvetica" size="12"> to 3. This line is not an essential part of the drag and drop operation. The user interface of a drag and drop is not as standard as other aspects of user interface; it can be useful to indicate visually that an object is the subject of a drag. In this example, we've widened the border around the swatch that's being dragged to provide a visual hint to the user during the drag and drop. We'll set </font><font face="Courier" size="11">the borderWidth</font><font face="helvetica" size="12"> back to 1 when the drag is over.</font></p>
<p></p>
<p>When the mouse enters an object during the drag, a <font face="Courier" size="11">dragEnter</font><font face="helvetica" size="12"> message is sent to the object. The </font><font face="Courier" size="11">dragEnter</font><font face="helvetica" size="12"> handler in this example, which is placed in the "ball" graphic, simply sets </font><font face="Courier" size="11">the acceptDrop</font><font face="helvetica" size="12"> property to true. The </font><font face="Courier" size="11">acceptDrop</font><font face="helvetica" size="12"> property determines whether the current object is "droppable". If the property is false, no </font><font face="Courier" size="11">dragDrop</font><font face="helvetica" size="12"> message is sent when the user releases the mouse button, so you can't drop. If the property is true, the </font><font face="Courier" size="11">dragDrop</font><font face="helvetica" size="12"> message is sent.</font></p>
<p></p>
<p><b><font face="Lucida Grande" size="11">Note:</font></b><font face="helvetica" size="12"> Revolution automatically sets </font><font face="Courier" size="11">the acceptDrop</font><font face="helvetica" size="12"> to true when dragging text between unlocked fields. If you want to be able to drop data onto any other kind of object, you need to set </font><font face="Courier" size="11">the acceptDrop</font><font face="helvetica" size="12"> to true when the mouse pointer enters the object.</font></p>
<p></p>
<p>When the user releases the mouse, a <font face="Courier" size="11">dragDrop</font><font face="helvetica" size="12"> message is sent to the graphic, triggering the </font><font face="Courier" size="11">dragDrop</font><font face="helvetica" size="12"> handler. In this example, the </font><font face="Courier" size="11">dragDrop</font><font face="helvetica" size="12"> handler retrieves the "text"&nbsp;element of </font><font face="Courier" size="11">the dragData</font><font face="helvetica" size="12"> (which we previously set in the </font><font face="Courier" size="11">mouseDown</font><font face="helvetica" size="12"> handler) and uses it to set the graphic's backgroundColor. The graphic now matches the color of the swatch the user dragged.</font></p>
<p></p>
<p>Finally, when the drag is finished, a <font face="Courier" size="11">dragEnd</font><font face="helvetica" size="12"> message is sent to the object that originated the drag and drop. In this case, that object is the swatch that the user dragged.</font></p>
<p></p>
<p>The <font face="Courier" size="11">dragEnd</font><font face="helvetica" size="12"> handler in this example sets </font><font face="Courier" size="11">the borderWidth</font><font face="helvetica" size="12"> of the dragged swatch back to 1. For simple drag and drop operations, you may not need to handle the </font><font face="Courier" size="11">dragEnd</font><font face="helvetica" size="12"> message at all.</font></p>
