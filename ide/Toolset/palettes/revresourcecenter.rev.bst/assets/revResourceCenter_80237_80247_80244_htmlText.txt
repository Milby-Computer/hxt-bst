<p><b><font face="helvetica" size="12">Discussion:</font></b></p>
<p>The <font face="Courier" size="11">mouseUp</font><font face="helvetica" size="12"> message is sent when the user clicks a field whose </font><font face="Courier" size="11">lockText</font><font face="helvetica" size="12"> property is set to true. When the </font><font face="Courier" size="11">mouseUp</font><font face="helvetica" size="12"> message is sent to the field, it triggers this </font><font face="Courier" size="11">mouseUp</font><font face="helvetica" size="12"> handler.</font></p>
<p></p>
<p>This handler animates the field's text so that it appears to scroll horizontally. It does this by first putting the text into the field with a number of leading spaces, then removing one space at a time so the text appears to move from right to left.</p>
<p></p>
<p><font color="#404040"><img src="202653" char=" "> </font><b><font face="Lucida Grande" size="11">Important!</font></b><font face="helvetica" size="12">  For this handler to work properly, the field's </font><font face="Courier" size="11">dontWrap</font><font face="helvetica" size="12"> property must be set to true. This prevents the text from wrapping to a second line. The field's </font><font face="Courier" size="11">textAlign</font><font face="helvetica" size="12"> property must also be set to "left" (the default). To tweak the vertical placement of the text in the field, set the field's </font><font face="Courier" size="11">fixedLineHeight</font><font face="helvetica" size="12"> property to true, then adjust its </font><font face="Courier" size="11">textHeight</font><font face="helvetica" size="12"> and </font><font face="Courier" size="11">textSize</font><font face="helvetica" size="12"> until you're pleased with the result.</font></p>
<p></p>
<p>The first thing this handler does is to place the text that will be animated in the field. This text consists of a number of spaces followed by the text that will be displayed: "You too can be a winner!". The leading spaces force the text to the right. Later, we'll remove the leading spaces one by one so that the text can move leftward.</p>
<p></p>
<p>The keyword <font face="Courier" size="11">me</font><font face="helvetica" size="12"> refers to the current object: the object whose script contains the handler that is executing. In this example, then, the text is placed in the field, because the </font><font face="Courier" size="11">mouseUp</font><font face="helvetica" size="12"> handler is in the field script.</font></p>
<p></p>
<p>The <font face="Courier" size="11">repeat</font><font face="helvetica" size="12"> loop performs the actual scrolling by removing the first character from the field each time through the loop. Since the text begins with a number of leading spaces, removing the first space has the effect of moving the rest of the text one space to the left. This is repeated until there are no more characters left in the field.</font></p>
<p></p>
<p>To the viewer, the text appears to come from the right side of the field and march toward the left. In reality, the handler is simply removing one character from the field during each iteration of the loop.</p>
<p></p>
<p><font color="#404040"><img src="202651" char=" "> </font><b><font face="Lucida Grande" size="11">Tip:</font></b><font face="helvetica" size="12">  Using this method, the scrolling text moves with a slight jerkiness because the spaces are wider than one pixel. The effect is very well-suited for simulating a teletype or a marquee display. To make text move smoothly across the screen, put the text in a field and use the move command to move the field across the screen.</font></p>
